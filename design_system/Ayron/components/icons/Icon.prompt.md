Inline SVG icon component using the Ayron (Lucide-based) icon set. Inherits `currentColor`, default 18px, 2px stroke.

```jsx
<Icon name="database" size={16} />
<Icon name="zap" size={18} stroke={2.2} />
```

Available names are listed in `IconProps["name"]`. Use 16px in dense UI, 17–18px in nav/buttons.
