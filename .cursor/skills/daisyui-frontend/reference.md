# daisyUI Component Reference

Quick lookup for class names. Docs: https://daisyui.com/components/

## Button
| Class | Purpose |
|-------|---------|
| btn | Base |
| btn-neutral, btn-primary, btn-secondary, btn-accent | Colors |
| btn-info, btn-success, btn-warning, btn-error | Semantic colors |
| btn-outline, btn-soft, btn-ghost, btn-link | Styles |
| btn-xs, btn-sm, btn-md, btn-lg, btn-xl | Sizes |
| btn-block, btn-wide, btn-square, btn-circle | Layout |

## Card
| Class | Purpose |
|-------|---------|
| card | Container |
| card-body | Content wrapper |
| card-title | Title |
| card-actions | Button row |
| card-border, card-dash | Border style |
| card-side | Horizontal layout |
| card-xs … card-xl | Size |
| image-full | Image overlay |

## Modal
| Class | Purpose |
|-------|---------|
| modal | Container (on dialog or div) |
| modal-box | Content area |
| modal-action | Action buttons |
| modal-backdrop | Click-to-close overlay |
| modal-toggle | Hidden checkbox (for no-JS) |
| modal-top, modal-middle, modal-bottom | Vertical position |
| modal-start, modal-end | Horizontal position |

## Form
| Class | Purpose |
|-------|---------|
| form-control | Wrapper |
| label, label-text, label-text-alt | Labels |
| input, input-bordered, input-ghost | Input |
| textarea, textarea-bordered | Textarea |
| select, select-bordered | Select |
| checkbox | Checkbox |
| toggle | Toggle switch |
| input-sm, input-lg | Sizes |

## Navbar
| Class | Purpose |
|-------|---------|
| navbar | Container |
| navbar-start | Left |
| navbar-center | Center |
| navbar-end | Right |

## Alert
| Class | Purpose |
|-------|---------|
| alert | Base |
| alert-info, alert-success, alert-warning, alert-error | Colors |
| alert-outline, alert-soft, alert-dash | Styles |
| alert-vertical, alert-horizontal | Layout |

## Dropdown
| Class | Purpose |
|-------|---------|
| dropdown | Container |
| dropdown-content | Menu container |
| dropdown-end, dropdown-top, etc. | Position |

## Menu
| Class | Purpose |
|-------|---------|
| menu | List |
| menu-horizontal | Row layout |
| menu-sm, menu-lg | Size |

## Badge
| Class | Purpose |
|-------|---------|
| badge | Base |
| badge-primary, badge-secondary, etc. | Colors |
| badge-outline | Outline |
| badge-sm, badge-xs | Size |

## Loading
| Class | Purpose |
|-------|---------|
| loading | Base |
| loading-spinner | Spinner |
| loading-dots, loading-ring | Variants |

## Semantic Colors (theme-aware)
| Class | Purpose |
|-------|---------|
| bg-base-100, bg-base-200, bg-base-300 | Backgrounds |
| bg-primary, bg-secondary, bg-accent | Accent backgrounds |
| text-base-content | Text |
| text-primary, text-secondary | Accent text |

## Themes
Set on `html`: `data-theme="light"`, `data-theme="dark"`, or built-in themes: cupcake, bumblebee, emerald, corporate, synthwave, retro, cyberpunk, valentine, halloween, garden, forest, aqua, lofi, pastel, fantasy, wireframe, black, luxury, dracula, cmyk, autumn, business, acid, lemonade, night, coffee, winter, dim, nord, sunset, etc.
